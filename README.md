# Container Consolidation in CaaS using Learning Automata

## 📌 Project Overview

This project focuses on intelligent container consolidation in a Container-as-a-Service (CaaS) cloud environment using Learning Automata and Q-Learning based resource allocation techniques.

The proposed system aims to optimize resource utilization, reduce makespan, improve throughput, and enhance virtual machine task scheduling efficiency in cloud computing environments.

---

## 🎯 Objectives

- Maximize resource utilization
- Reduce makespan and execution time
- Improve VM task allocation efficiency
- Optimize container migration and consolidation
- Enhance cloud performance using Learning Automata

## ⚙️ Proposed Methodology

The system uses Learning Automata and Q-Learning algorithms for intelligent task scheduling and container placement in cloud environments.

The framework performs:
- Resource monitoring
- VM allocation
- Task scheduling
- Container migration
- Load balancing
- Performance evaluation

The proposed model evaluates system performance using:
- Makespan
- Throughput
- CPU Utilization
- Energy Consumption
- Success Ratio

---

## 📊 System Workflow

1. Task generation
2. VM selection using Q-Learning
3. Task allocation and processing
4. Resource utilization monitoring
5. Performance analysis and optimization
